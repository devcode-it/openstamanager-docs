# ➕ Creazione

Per creare un impianto in OpenSTAManager si dovrà cliccare sul tasto (+).

La schermata che si presenterà sarà questa, dove sarà possibile inserire:

* Matricola
* Nome
* Cliente
* Sede
* Tecnico assegnato
* Categoria
* Sottocategoria
