# ➕ Creazione

{% hint style="info" %}
Il modulo **Gestione documentale** presenta la possibilità di gestire una serie di documenti attraverso la compilazione di alcuni campi.
{% endhint %}

Per creare un nuovo documento si deve premere il tasto (+), e inserire:

* Nome
* Categoria
* Data
