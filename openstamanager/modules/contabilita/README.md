# ⚖ Contabilità

## ⛵ Navigazione

Il modulo è raggiungibile dal menu laterale del gestionale, sotto il link Contabilità.

![](<../../../.gitbook/assets/image (563).png>)

## 🖥️ Interfaccia

Nel menu a tendina è possibile trovare i moduli:

* Stampe contabili
* Prima nota
* Piano dei conti
* Scadenzario

&#x20;                                                         ![](<../../../.gitbook/assets/image (560).png>)
