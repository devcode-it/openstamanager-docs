# Stati di attività

{% hint style="info" %}
Il modulo **Stati di attività** offre di base 4 stati, assegnabili ad un tecnico,quali:
{% endhint %}

* TODO(Da programmare)
* FAT(Fatturato)
* OK(Completato)
* WIP(Programmato)

## Navigazione

Il modulo **Stati di attività** è raggiungibile attraverso il menu laterale del gestionale, sotto il link **Attività**.
